Burrito: A Marriage of Taxonomy and Function

A tasty burrito.

This is a tool to visualize the relationship between taxonomy and function. You can view sample data or upload your own data.

Upload your own data

Select this option if you have your own data to upload and visualize.
16s Reads You only have 16s reads. Functional Contributions You have run PICRUST yourself and have reads and contributions. Genomic Annotations You have the genome content of each taxa and their abundances.
Genome Abundances A file containing abundances of each genome in each sample. See documentation for an example.
Taxonomic Abundances A file containing abundances of each taxa in each sample. See documentation for an example.
16s Reads A file containing abundaces of each taxa in each sample. See documentation for an example.
Genome Annotations A file containing the functional content of each genome. See documentation for an example.
Function Contributions A file containing the functional contribution of each taxa. See documentation for an example.
Optional Files:
Taxonomic Hierarchy A file containing all taxa in your abundance file, in a hierarchy showing their relatedness. See documentation for a format example. Uses greengenes by default if no file uploaded. Function Hierarchy A file containing the functions in your contribution or genome content file, in a hierarchy showing their relatedness. See documentation for a format example. Uses BRITE by default if no file is uploaded. Sample Map A file describing the relationship between samples, i.e. which ones are the same treatment condition.

Visualization Options

Select the lowest level of taxonomic detail to be displayed. Going to the OTU level will significantly slow down the program.
Random color scale will help discern closely related taxa but hierarchical color scale will look more pleasing.
