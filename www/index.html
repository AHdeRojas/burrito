Burrito
Burrito is a tool for visualizing the relationship between microbiome taxonomy and function. A detailed description of the input parameters and options is available in the user manual. Source code is available here

Taxonomic Data Manual>Taxonomy
Taxonomic abundance table: Upload a taxonomic abundance table
(can be raw read counts or relative abundances).
Taxonomy:
Greengenes taxonomy (default; compatible with PICRUSt)
Custom taxonomy:
Choose a taxonomy for hierarchically displaying OTU relationships.
Minimum taxonomic resolution: Choose the minimum taxonomic resolution to view
(lower resolution may improve performance for large datasets).
Functional Data Manual>Function
Metagenomics-based function abundances:
No metagenomic function abundances available (default)
Metagenomic function abundance table:
Choose whether you will be providing metagenomics-based functional data
to compare to the taxonomy-based function abundances.
Taxonomy-function linking method:
Run PICRUSt (default)
Genomic content table:
Function contribution table:
Choose the method for linking taxonomic abundances to metagenomic content.
Function hierarchy:
BRITE function hierarchy (default, compatible with PICRUSt)
Custom function hierarchy:
Choose a functional hierarchy for hierarchically displaying function relationships.
Minimum function resolution:
Choose the minimum function resolution to view
(lower resolution may improve performance for large datasets).
Sample grouping Manual>Grouping
Sample labels:
No sample labels (default) Sort samples alphabetically
Sample labels:
Choose whether you will be providing sample labels.
Group by label:
Choose the label to be used for grouping samples.
Visualization Manual>Visualization
Color scheme:
Choose the type of color scale used during display.
